「Webコンテンツ作成」の成果

本テーマについて

情報メディア実験では、実験やプレゼンテーションを通じて実践的な能力を養い、コミュニケーションスキルを強化し、モーション解析やWebページ作成などのスキルを身につけ、リーダーシップと解決能力を高めます。

プロジェクト課題について

このプロジェクト課題は非常に大変で、情報メディア実験の授業が嫌いになりそうでした。授業がどんどん先に進む中で、一つ前の課題が終わらないまま作業を続けることが多く、現在どの部分をやっているのか分からなくなることがありました。さらに、周りの人たちが簡単にこなしているように見える課題が自分にはうまくできず、本当に苦労しました。この経験はとてもストレスで、授業に対するモチベーションを保つのが難しかったですが、達成感はどの課題よりも感じることができました。

プロジェクト課題

数当てゲーム

私は数当てゲームを作成する情報メディア実験の授業では、比較的に楽に作業を進めることができました。しかし、これは決して簡単な作業ではありませんでした。授業中、隣の席の人や周りの友達が自分よりも先に進んでいる講義資料を見ながら作業している姿を目にすることが多く、その度に焦りを感じました。また、講義資料のどの部分を参照すればいいのかわからなくなることも頻繁にあり、そのたびに作業が停滞しました。なので他の授業の課題と比較すると、この数当てゲームの制作は非常に手間がかかりました。結果的には数当てゲームを作成して大変でしたが、JavaScriptはJavaと似ていたこともありコードも覚えやすくパズルみたいに作業することができたため楽しかったです。

各授業回の学習事項

第一回 WWW と HTML

第一回の情報メディア実験の授業ではこれからの情報メディア実験の概要とHTMLの基本のタグやブロック要素、インデント要素などを実践を用いいて学び、HTMLの構成の仕方などHTMLのデバックの仕方も学ぶことができました。

演習の成果: 演習1-4:HTMLのさまざまな要素

第二回 CSS による Web ページの装飾

第二回の情報メディア実験の授業ではCSS による見た目の設定、CSS の全体像、CSSのデバッグなどcss言語に関わることを全般に学びました。css言語ではいろんな要素があるのでこれからmbn web docsを使って上達していきたいです。

演習の成果: 演習2-3:ウェブブラウザの紹介

第三回 JavaScript の基本

第三回の情報メディア実験の授業ではjavaScriptにできること、ページに JavaScript を追加する方法、JavaScript と Java の違いなどを学びました。JavaとJavaScriptは似ているので覚えやすいですが、その分似ているためケアレスミスが多発しそうなので違いをJavaとJavaScriptの違いを認識して覚えていきたいです。

演習の成果: 演習3-6:数当てゲーム

第四回 DOM と JavaSclipt

第四回の情報メディア実験の授業では、DOMがWebコンテンツを表現するためのデータ構造であることを学びました。DOMを使用することで、JavaScriptなどのプログラムを用いて、Webコンテンツの内容を読み取ったり、コンテンツに変更を加えたりする処理を実装できるようになります。この授業を通じて、Web開発におけるDOMの重要性とその操作方法について理解を深めることができました。

演習の成果: 演習4-2~5:練習4-2から4-5

第五回 イベント処理

第五回の情報メディア実験の授業ではイベントとは何か、イベント処理の例、イベントの種類、input要素などを学びました。プロジェクト課題をやっていくなかでとても大切な部分なので、しっかり復習しておぼえていきたいです。

演習の成果: 演習5-3:イベントの種類

工夫点，苦心点，強調点，感想

工夫点

この「Webコンテンツ作成の成果」において、デザインに統一性を保つことを重視しました。HTMLとCSSの授業で学んだ知識を活かし、全体の見た目やレイアウトを統一するために、フォントの選定やカラーリングに一貫性を持たせることで、ユーザーが違和感なくコンテンツを閲覧できるように工夫しました。

苦心点

JavaScriptはJavaに似ている部分もありましたが、その違いを理解するのに苦労しました。特に、DOM操作やイベント処理に関する部分では、何度もエラーが発生し、その都度デバッグに時間を取られました。結果として、エラーの原因を特定し、修正するための スキルを大幅に向上させることができました。 授業が進む中で、複数の課題を同時に進めなければならない場面が多くありました。特に、プロジェクト課題の進捗を管理するのが難しく、一つの課題に集中していると他の課題が遅れてしまうことがありました。

強調点

プロジェクトを進めるにあたり、周囲の友達やseの方と積極的にコミュニケーションを取り、助け合いながら進めることができました。特に、わからないことがあった時には周囲の友達やseに相談し、逆に自分が得意な部分では他の仲間をサポートするように心がけました。これにより、課題を最終的に終えることができました。 授業で学んだことだけでなく、独自にHTMLとCSS学習を進めました。この自己学習のおかげで最初の授業の課題は期限内にてしゅつできたと感じています。

感想

このプロジェクト課題を通じて、Webコンテンツ作成に関するスキルを大幅に向上させることができました。特に、HTML、CSS、JavaScriptを用いたWebページの作成は初めての経験でしたが、各授業で学んだ知識を総合的に活用することで、実践的なスキルを身につけることができました。また、プロジェクトを進める中で直面した様々な課題や困難を乗り越えることで、問題解決能力やコミュニケーションの重要性を実感しました。これらの経験は、今後の学習や就職した後においても大いに役立つと感じています。最後に、このプロジェクトを通じて得た達成感は非常に大きく、自己成長を強く実感することができました。今後も引き続き、Web開発のスキルを磨き続け、多くの人に役立つコンテンツを提供できるよう努力していきたいと思います。さらに、新しい技術に積極的に挑戦し続けることで、常に最新のトレンドを追い、自己研磨に励みたいと思います。
